## Requisitos

- [Node.js](https://nodejs.org/) instalado en tu sistema.

### Instalación

```
npm install
```

### Run

```
npm run dev
```

Abre tu navegador y visita http://localhost:3000.

Disfruta de la aplicación de chat en tiempo real.
